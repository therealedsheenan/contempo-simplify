# contempo-simplify
Simplified version of contempo

### Development Server
```
yarn dev
```
or 
```
npm run dev
```

### Production
```
yarn prod
```
or 
```
npm run prod
```

### Run tests
```
yarn test
```
or 
```
npm test
```

### Further Help

See the [contempo](https://github.com/therealedsheenan/contempo) repository for more details.
